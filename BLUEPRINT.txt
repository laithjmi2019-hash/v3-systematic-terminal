## 🔢 1. NORMALIZATION SYSTEM

```
score = clamp((value - min) / (max - min), 0, 1)
```

### Anchors:

| Metric        | Min | Max |
| ------------- | --- | --- |
| ROE           | 0%  | 30% |
| Net Margin    | 0%  | 25% |
| ROA           | 0%  | 15% |
| Gross Margin  | 20% | 80% |
| Current Ratio | 1   | 3   |
| Debt/Equity   | 0   | 2   |
| P/E           | 10  | 40  |
| P/FCF         | 8   | 30  |

---

## 📉 2. VOLATILITY PENALTY

```
VolatilityPenalty = min(std_dev / mean, 0.5)

AdjustedScore = BaseScore × (1 - VolatilityPenalty)
```

* Margins
* Revenue growth
* Free cash flow

# 🧠 PHASE 4 — ADVANCED METRICS

---

## 1. Earnings Quality (Accruals)

```
Accruals = (Net Income - FCF) / Total Assets
```

* > 10% → Warning
* > 20% → Critical

---

## 2. Historical vs Forward PE

```
Historical PE = avg(Price / EPS over 5 years)
```

* Forward PE > Historical + 20% → Over-optimism
* Forward PE < Historical → Undervalued

---

## 3. Capital Allocation Score

* Share dilution trend
* Buybacks
* ROE consistency

---

## 4. Score Momentum

```
ScoreMomentum = CurrentScore - Score_1_year_ago
```

---

## 5. Data Quality Score

```
DataQuality = valid_metrics / total_metrics
```

* > 90% → Reliable
* 70–90% → Moderate
* <70% → Low

---

## 6. Dynamic Score Cap

If any pillar < 20%:

```
MaxScoreCap = 30 + (StrongPillars × 5)
FinalScore = min(FinalScore, MaxScoreCap)
```
